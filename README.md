# 🌦️ weatherApp

A beautiful, minimal weather app powered by free open-source APIs and Claude AI for atmospheric weather summaries.

![Weather App Screenshot](screenshot.png)

## ✨ Features

- 🔍 **Search any city** worldwide using free geocoding
- 🌡️ **Current conditions** — temperature, feels-like, humidity, wind speed, UV index
- 📅 **5-day forecast** with daily highs and lows
- 🤖 **AI-generated summaries** via Claude (optional — requires Anthropic API key)
- ✨ **Animated starfield** background with glassmorphism UI
- 📱 **Responsive design** — works on mobile and desktop
- ⚡ **No backend required** — fully client-side HTML/CSS/JS

## 🚀 Getting Started

### Option 1 — Just open it

This app is a single HTML file. No build step, no dependencies to install.

```bash
# Clone the repo
git clone https://github.com/your-username/weather-app.git
cd weather-app

# Open in browser
open index.html
```

### Option 2 — Serve locally

```bash
# Python 3
python -m http.server 8080

# Node.js (npx)
npx serve .
```

Then visit `http://localhost:8080`.

## 🔑 Enabling AI Summaries (Optional)

AI-powered weather summaries are generated by Claude. To enable them:

1. Get a free API key at [console.anthropic.com](https://console.anthropic.com)
2. Open `index.html` and find this line near the top of the `<script>` section:

```js
const ANTHROPIC_API_KEY = 'YOUR_ANTHROPIC_API_KEY_HERE';
```

3. Replace `YOUR_ANTHROPIC_API_KEY_HERE` with your actual key and save.

> **Note:** If no key is set, the app works fully — AI summaries are simply hidden. All weather data still loads normally.

## 🌐 APIs Used

| API | Purpose | Cost |
|-----|---------|------|
| [Open-Meteo Geocoding](https://open-meteo.com/en/docs/geocoding-api) | City name → coordinates | Free |
| [Open-Meteo Forecast](https://open-meteo.com/en/docs) | Weather data | Free |
| [Anthropic Claude](https://console.anthropic.com) | AI weather summaries | Optional (paid) |

No API keys are required for weather data — only Claude summaries need a key.

## ⚠️ Security Note

> Do **not** commit your Anthropic API key to a public repository. If you plan to deploy this publicly, move the API call to a backend proxy or use environment variables.

Consider adding your key to `.gitignore` via a config file:

```bash
echo "config.js" >> .gitignore
```

And loading it externally:

```html
<script src="config.js"></script> <!-- contains: const ANTHROPIC_API_KEY = '...'; -->
```
